## 1. 관리자 접속 방법

1. 웹브라우저(Chrome, Safari 등)를 엽니다.
2. 주소창에 **관리자 주소**를 입력합니다.  
   예: `https://www.고객도메인.com/adm`  
   (실제 주소는 제작 시 별도로 전달됩니다.)
3. **아이디**와 **비밀번호**를 입력하고 로그인합니다.
4. 비밀번호는 **처음 로그인 후 반드시 변경**하는 것을 권장합니다.

> 관리자 계정은 **공유하지 마세요.** 꼭 필요한 담당자만 사용하세요.

---

## 2. 관리자 기본 화면

로그인 후 왼쪽(또는 상단) 메뉴에서 자주 쓰는 항목:

| 메뉴 | 하는 일 |
|------|---------|
| **환경설정** | 사이트 제목, 기본 설정 |
| **게시판관리** | 공지·소식·갤러리 등 게시판 만들기·수정 |
| **회원관리** | 회원 목록 (회원 기능 사용 시) |
| **메뉴관리** | 상단 메뉴 이름·링크 |
| **내용관리** | 약관·고정 페이지 (일부 사이트) |

홈페이지 **디자인(색·레이아웃)** 은 제작사가 맞춰 둔 경우가 많습니다.  
일반적으로 **글·메뉴·이미지** 위주로 관리합니다.

---

## 3. 메뉴 수정 방법

상세 링크 형식·체크리스트는 제작사 제공 **[MENU-GUIDE.md](MENU-GUIDE.md)** 또는 [docs/client/site-operation-guide-template.md](docs/client/site-operation-guide-template.md)를 참고하세요.

**주의 (고객이 직접 수정할 때)**

- **PC용·모바일용** 메뉴를 둘 다 확인·저장 (한쪽만 수정하면 모바일과 다를 수 있음)
- 링크는 `/page/contact.php`, `/bbs/board.php?bo_table=notice` 형태 — **도메인을 중복**해 넣지 않기
- 외부 사이트는 `https://` 포함, 새 창 여부 확인
- 게시판 메뉴는 관리자 **게시판 ID**와 URL의 `bo_table`이 **같아야** 함
- 문의 메뉴와 상단·하단 **상담 버튼** URL이 같은지 확인
- 개인정보처리방침(`/page/privacy.php`) · FAQ(`/bbs/board.php?bo_table=faq`) 링크 유지

1. 관리자 → **환경설정** → **메뉴설정** (또는 **메뉴관리**)
2. **PC용 메뉴**, **모바일용 메뉴**를 각각 확인합니다.
3. 수정할 메뉴를 선택합니다.
   - **메뉴명** — 화면에 보이는 글자
   - **링크** — 클릭 시 이동할 주소 (게시판, 페이지 URL)
4. **새 메뉴**는 「추가」로 등록합니다.
5. **순서**는 위·아래 버튼 또는 순서 번호로 조정합니다.
6. **저장** 후 홈페이지에서 새로고침하여 확인합니다.

---

## 4. 게시판 글 작성 방법

1. 홈페이지에서 해당 메뉴(예: 공지사항)를 클릭하거나, 관리자에서 게시판으로 이동합니다.
2. **글쓰기** 버튼을 클릭합니다.
3. **제목**을 입력합니다.
4. **본문**에 내용을 입력합니다. (굵게, 목록 등은 에디터 버튼 사용)
5. 필요하면 **이미지·파일 첨부**합니다.
6. **공지글** — 상단에 고정하려면 「공지」 옵션을 선택합니다.
7. **비밀글** — 본인만 보려면 「비밀글」을 선택합니다.
8. **저장** 또는 **등록**을 클릭합니다.

글 **수정·삭제**는 글 보기 화면의 「수정」「삭제」 버튼을 사용합니다.

---

## 5. 이미지 올리는 방법

| 권장 | 설명 |
|------|------|
| **크기** | 너무 큰 파일(수 MB)은 업로드·로딩이 느려집니다. 웹용으로 **가로 1920px 이하** 정도를 권장합니다. |
| **파일명** | 한글·공백보다 `news-2026-01.jpg` 처럼 **영문·숫자**가 안전합니다. |
| **확인** | 저장 후 **PC와 휴대폰**에서 잘리지 않는지 확인합니다. |

메인 큰 배너 이미지는 제작사가 `img/main` 폴더에 넣어 둔 경우가 있습니다.  
교체 방법은 **제작사에 문의**하세요.

---

## 6. 유튜브 게시판 사용 방법

영상 전용 게시판(유튜브 스킨)을 사용하는 경우:

1. 유튜브에서 영상 **공유** → **링크 복사**
2. 글쓰기 화면에서 **여분필드 `wr_1`** 또는 안내된 **「유튜브 URL」** 칸에 붙여넣기
3. 지원 주소 예:
   - `https://www.youtube.com/watch?v=영상ID`
   - `https://youtu.be/영상ID`
   - `https://www.youtube.com/shorts/영상ID`
4. **저장** 후 **글 보기**에서 썸네일·재생이 되는지 확인합니다.

주소가 잘못되면 **대체 이미지**만 보일 수 있습니다. 링크를 다시 확인하세요.

---

## 7. 공지사항 관리

- **공지글** — 「공지」로 등록하면 목록 **맨 위**에 표시됩니다 (공지 스킨 사용 시).
- **공지 해제** — 수정 화면에서 공지 옵션을 해제합니다.
- 중요한 안내는 **공지 + 짧은 제목**으로 작성하면 찾기 쉽습니다.

---

## 8. 문의 확인 방법

**문의·상담 게시판**을 쓰는 경우:

1. 관리자 또는 해당 게시판 목록에서 **새 글**을 확인합니다.
2. **비밀글**로 올라온 문의는 **관리자·작성자만** 볼 수 있습니다.
3. 연락처(전화, 이메일)는 **개인정보**이므로 외부에 공유하지 마세요.
4. 답변은 **게시판 답글** 또는 **전화·이메일**로 처리합니다.

하단 **「상담문의」** 버튼·모달로 접수하면 **완료 안내 페이지**로 이동할 수 있습니다. (제작사 설정)

- **이메일·텔레그램 알림** — 제작사가 `_site.config.php`에서 설정합니다. 알림이 오지 않아도 게시판에 글이 있으면 확인 가능합니다.
- **광고 전환 추적** — 완료 페이지에만 코드가 들어갈 수 있습니다. 변경은 제작사에 문의하세요.

---

## 9. SEO 글 작성 기본 팁

검색에 도움이 되도록:

- **제목**에 핵심 키워드를 자연스럽게 넣습니다.
- **첫 문단**에 글 주제를 분명히 씁니다.
- 이미지에는 **설명(alt)** 을 넣을 수 있으면 넣습니다.
- **너무 짧은 글**(한두 줄)만 반복하지 않습니다.
- 다른 사이트 글을 **그대로 복사**하지 않습니다.

## 10. 주의사항 (꼭 읽어 주세요)

| 하지 말아야 할 일 | 이유 |
|-------------------|------|
| 서버·FTP에서 **임의 파일 삭제** | 사이트가 멈출 수 있습니다 |
| `bbs`, `lib`, `adm` 폴더 **수정** | 프로그램이 망가질 수 있습니다 |
| **DB·비밀번호** 메일·메신저로 공유 | 보안 사고 위험 |
| **관리자 계정** 여러 사람이 동시 사용 | 실수·해킹 위험 |
| 고객 **개인정보** SNS·외부 공유 | 법적 문제 |

디자인·기능 **대규모 변경**은 제작사에 요청하세요.

---

## 11. 문제가 생겼을 때

제작사·유지보수 담당자에게 아래를 알려 주시면 빠릅니다.

1. **오류가 난 주소(URL)** — 주소창 전체 복사
2. **화면 캡처** — 오류 메시지가 보이게
3. **무엇을 하다가** 발생했는지 (예: 글 저장, 이미지 업로드)
4. **브라우저** — Chrome, Safari, Edge 등
5. **PC인지 휴대폰인지**
