자동댓글 배포 패키지 (버전 1.1.1)

[설치 방법 — 신규·기존 사이트 동일]
1) 이 폴더 안의 plugin, extend 를 그누보드 루트(public_html 등)에 그대로 덮어씁니다.
   예) plugin/auto_comment → /plugin/auto_comment
       extend/auto_comment.extend.php → /extend/auto_comment.extend.php
2) 브라우저에서 아래 중 하나를 최고관리자로 1회 실행합니다.
   - 처음 설치: /plugin/auto_comment/install.php
   - 이미 사용 중(덮어쓰기만 한 경우): /plugin/auto_comment/update.php
     (기존 설정, API 키, 게시판 설정, 예약목록은 유지됩니다.)

※ 별도 업데이트 전용 패키지 없이, 항상 이 full 패키지로 덮어쓰면 됩니다.
