## 1. 안내 문구

이 문서는 **샘플 홈페이지**를 운영하시는 **관리자·담당자**를 위한 안내서입니다.

- 관리자 **아이디·비밀번호**는 **이 문서에 적지 않고**, 별도로 안전하게 전달받아 보관해 주세요.
- **최초 로그인 후 비밀번호 변경**을 권장합니다.
- 사이트 구조·디자인·프로그램 설정 변경이 필요하면 **온오프마케팅**에 문의해 주세요.

## 3. 관리자 로그인 방법

1. 웹 브라우저(Chrome, Edge, Safari 등)를 엽니다.
2. 주소창에 **https://example.com/adm** 을 입력합니다.
3. 전달받은 **아이디**와 **비밀번호**를 입력하고 **로그인**합니다.
4. 로그인 후 관리자 화면이 보이면 성공입니다.
5. **비밀번호 변경**을 권장합니다.

### 로그인이 안 될 때

1. 주소·`https`·`www` 포함 여부를 확인합니다.
2. 아이디·비밀번호를 다시 확인합니다.
3. **온오프마케팅(010-0000-0000)** 에 연락합니다.

---

## 4. 관리자 메뉴 기본 설명

| 메뉴 | 하는 일 |
|------|---------|
| **환경설정** | 사이트 제목, 기본 정보 |
| **메뉴관리** | 상단 메뉴 이름·링크·순서 |
| **게시판관리** | 공지, 소식, FAQ, 갤러리 등 |
| **회원관리** | 회원 목록 (회원 기능 사용 시) |
| **내용관리** | 약관·고정 페이지 |
| **게시글 관리** | 게시판별 글 목록 |

---

## 5. 메뉴 수정 방법

1. 관리자 로그인 → **메뉴관리**
2. PC·모바일 메뉴 각각 확인
3. 메뉴명·링크 수정 또는 메뉴 추가
4. 순서 조정 후 **저장**
5. https://example.com 에서 PC·휴대폰 확인

---

## 6. 게시판 글 작성 방법

1. 해당 메뉴 또는 게시판 접속
2. **글쓰기** 클릭
3. **제목**·**본문** 입력
4. 이미지·파일 첨부 (필요 시)
5. 공지·비밀글 옵션 선택 (필요 시)
6. **저장** 후 화면 확인

---

## 7. 게시글 수정·삭제 방법

- **수정:** 글 열기 → 수정 → 저장  
- **삭제:** 글 열기 → 삭제 (복구 어려움)

---

## 8. 이미지 업로드 방법

| 용도 | 권장 |
|------|------|
| 본문 이미지 | 가로 1200px 이하 |
| 썸네일 | 가로 600px 이하 |
| 용량 | 1MB 이상이면 압축 |

- 파일명: `news-2026-01.jpg` (영문·숫자 권장)
- 한글·공백 파일명은 피하기

---

## 9. 유튜브 게시판 사용 방법

1. 유튜브 **링크 복사**
2. 글쓰기 **유튜브 URL** 칸에 붙여넣기
3. 제목·설명 입력 후 저장
4. 목록·글 보기에서 재생 확인

지원 예: `watch?v=`, `youtu.be/`, `shorts/`

---

## 10. FAQ 게시판 사용 방법

- **질문** → 제목  
- **답변** → 본문  
- 분류 선택 → 저장 → 목록에서 아코디언 확인

---

## 11. 문의 확인 방법

1. 문의 게시판 목록 확인
2. 이름·연락처·내용 확인
3. 상태 변경 (신규 → 확인중 → 상담완료 등)
4. 개인정보 외부 공유 금지
