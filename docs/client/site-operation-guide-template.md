## 3. 관리자 로그인 방법

1. 웹 브라우저(Chrome, Edge, Safari 등)를 엽니다.
2. 주소창에 **관리자 URL**을 입력합니다.  
   → {{ADMIN_URL}}
3. 전달받은 **아이디**와 **비밀번호**를 입력하고 **로그인**합니다.
4. 로그인 후 관리자 화면(대시보드)이 보이면 성공입니다.
5. **환경설정** 등에서 **비밀번호를 변경**해 두는 것을 권장합니다.

### 로그인이 안 될 때

1. 주소가 `{{ADMIN_URL}}`과 같은지 확인합니다. (`http` / `https`, `www` 유무)
2. 아이디·비밀번호 대소문자·띄어쓰기를 다시 확인합니다.
3. 여전히 안 되면 **{{AGENCY_NAME}}**({{SUPPORT_CONTACT}})에 연락합니다.

> **주의:** 관리자 비밀번호를 이메일·메신저·PDF에 넣어 여러 사람과 공유하지 마세요.

## 4. 관리자 메뉴 기본 설명

|------|---------|

| **메뉴관리** | 상단·하단 메뉴 이름, 링크, 순서 |

## 5. 메뉴 수정 방법

1. 관리자에 로그인합니다.
2. **환경설정** → **메뉴설정** 또는 **메뉴관리**로 이동합니다.
3. **PC용 메뉴**, **모바일용 메뉴**를 각각 확인합니다.
4. 수정할 메뉴를 선택합니다.  
   - **메뉴명** — 화면에 보이는 이름  
   - **링크** — 클릭 시 이동할 주소(게시판, 페이지)
5. **새 메뉴**는 「추가」로 등록합니다.
6. **순서**는 위·아래 버튼 또는 순서 번호로 조정합니다.
7. **저장** 후 홈페이지({{DOMAIN}})에서 **PC와 휴대폰** 모두 새로고침하여 확인합니다.

## 6. 게시판 글 작성 방법

1. 홈페이지에서 해당 메뉴(예: 공지사항)를 클릭하거나, 관리자에서 해당 게시판으로 이동합니다.
2. **글쓰기** 버튼을 클릭합니다.
3. **제목**을 입력합니다.
4. **본문**에 내용을 입력합니다. (굵게, 목록 등은 에디터 버튼 사용)
5. 필요하면 **이미지 삽입**·**파일 첨부**합니다.
6. **공지글** — 목록 상단에 고정하려면 「공지」 옵션을 선택합니다.
7. **비밀글** — 본인·관리자만 보려면 「비밀글」을 선택합니다.
8. **저장** 또는 **등록**을 클릭합니다.
9. 저장 후 **목록·글 보기** 화면에서 내용이 올바른지 확인합니다.

## 7. 게시글 수정·삭제 방법

### 수정

1. 수정할 **게시글**을 엽니다.
2. **수정** 버튼을 클릭합니다.
3. 제목·본문·첨부 등을 고칩니다.
4. **저장** 후 화면을 다시 확인합니다.

### 삭제

1. 글 보기 화면에서 **삭제**를 클릭합니다.
2. 확인 메시지에 동의하면 삭제됩니다.

> **주의:** 삭제한 글은 **복구가 어렵습니다.** 삭제 전 내용·첨부가 필요하면 백업해 두세요.

## 8. 이미지 업로드 방법

### 권장 기준

| 용도 | 권장 크기 |
|------|-----------|
| 일반 본문·소식 이미지 | 가로 **1200px 이하** |
| 목록 썸네일 | 가로 **600px 이하** |
| SNS·검색 미리보기(OG) | **1200×630** (제작사 설정) |
| 파일 용량 | **1MB 이상**이면 압축 후 업로드 |

### 업로드 시 주의

1. **파일명**은 `news-2026-01.jpg`처럼 **영문·숫자·하이픈(-)** 을 권장합니다.
2. **한글 파일명·공백**은 오류·깨짐이 날 수 있어 피합니다.
3. 에디터 **이미지 삽입** 또는 **파일 첨부**로 올립니다.
4. 저장 후 **PC·휴대폰**에서 잘리지 않는지 확인합니다.

대표 이미지·메인 배너가 필요한 게시판은 제작 시 안내된 방식을 따르고, 메인 화면 큰 이미지 교체는 **{{AGENCY_NAME}}**에 문의하세요.

## 9. 유튜브 게시판 사용 방법

영상 전용 게시판(유튜브 스킨)을 사용하는 경우입니다.

1. 유튜브에서 영상 **공유** → **링크 복사**
2. 글쓰기 화면 **「유튜브 URL」**(여분필드 `wr_1`)에 붙여넣기
3. **제목**·**영상 설명**(있는 경우) 입력
4. **저장** 후 **목록·글 보기**에서 썸네일·재생 확인

### 지원 URL 예시

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/shorts/VIDEO_ID`

### 주의

- **비공개·제한된 영상**은 사이트에 표시되지 않을 수 있습니다.
- **잘못된 URL**이면 썸네일·영상 대신 안내 문구만 보일 수 있습니다.

## 10. FAQ 게시판 사용 방법

FAQ 전용 스킨(`faq-accordion`)을 사용하는 경우입니다.

1. **질문** → 제목(`wr_subject`)에 입력
2. **답변** → 본문(`wr_content`)에 입력
3. **분류**가 있으면 해당 분류를 선택
4. **저장** 후 목록에서 **아코디언**(질문 클릭 시 답 펼침)으로 표시되는지 확인

관리자에서 게시판 **「목록에서 내용 사용」**이 켜져 있어야 목록에 답변이 바로 보일 수 있습니다. (설정은 {{AGENCY_NAME}}에 문의)

## 11. 문의 확인 방법

문의·상담 게시판(`inquiry`, `landing-inquiry` 등)을 사용하는 경우입니다.

1. 관리자 또는 해당 **문의 게시판 목록**에 접속합니다.
2. **신규 문의** 글을 확인합니다.
3. **이름·연락처·이메일·문의 내용**을 확인합니다.
4. 필요 시 **상태**(신규 → 확인중 → 상담완료 등)를 변경합니다.
5. **담당자·관리 메모** 필드가 있으면 내부용으로 기록합니다.

### 문의 상태 예시

| 상태 | 의미 |
|------|------|
| 신규 | 방금 접수됨 |
| 확인중 | 담당자 확인·연락 중 |
| 상담완료 | 상담 종료 |
| 계약완료 | 계약까지 완료 |
| 보류 | 추후 연락 예정 |
| 스팸 | 광고·스팸으로 분류 |

> **개인정보:** 문의 내용에는 이름·전화번호 등이 포함될 수 있습니다. **외부 공유·SNS 게시 금지**, 불필요한 보관은 피하세요.

## 12. 공지사항 관리 방법

1. 공지 게시판에서 **글쓰기**를 클릭합니다.
2. 제목·본문을 작성합니다.
3. **「공지」** 옵션을 선택하면 목록 **상단**에 고정되는 경우가 많습니다.
4. 공지를 해제하려면 **수정** 화면에서 공지 옵션을 끕니다.
5. 중요한 안내는 **짧고 분명한 제목**으로 작성합니다.

## 13. 팝업·배너·CTA 관리 안내

- 메인 **팝업**, **하단 플로팅 버튼**, **큰 배너 문구**는 제작 시 코드로 연결된 경우가 많습니다.
- 문구·이미지 변경이 필요하면 **{{AGENCY_NAME}}**에 요청하는 것을 권장합니다.
- **FTP에서 파일을 임의로 수정**하지 마세요. 오류·보안 문제가 생길 수 있습니다.
- 이벤트 요청 시 **바꿀 문구**, **기간**, **링크 URL**, **참고 캡처**를 함께 전달해 주세요.

## 14. SEO 글 작성 기본 팁

검색·신뢰도에 도움이 되도록 다음을 참고합니다.

1. **제목**에 핵심 키워드(서비스명·지역명 등)를 자연스럽게 넣습니다.
2. **첫 문단**에 글 주제를 분명히 씁니다.
3. 다른 사이트 글을 **그대로 복사**하지 않습니다.
4. 이미지에는 가능하면 **설명(대체 텍스트)** 을 넣습니다.
5. **너무 짧은 글**만 반복하지 않습니다.
6. **지역명·서비스명**을 과장 없이 포함합니다.
7. 「최고」「유일」「100%」 등 **과장·허위 표현**은 피합니다.

5. 사이트 하단 **「개인정보처리방침」** 내용을 확인하고, 회사 정보·책임자·시행일이 맞는지 점검합니다.

## 16. 자주 발생하는 문제와 해결 방법

| 증상 | 확인할 것 |
|------|-----------|
| **로그인이 안 됨** | URL·아이디·비밀번호·Caps Lock → {{AGENCY_NAME}} 문의 |
| **이미지가 안 올라감** | 파일 크기·용량·파일명(영문)·브라우저 새로고침 |

| **문의 알림이 안 옴** | 게시판에 글이 있는지 먼저 확인 (알림은 제작사 설정) |
| **모바일 화면이 이상함** | PC에서 저장한 HTML·큰 이미지·캐시 삭제 후 재확인 |
